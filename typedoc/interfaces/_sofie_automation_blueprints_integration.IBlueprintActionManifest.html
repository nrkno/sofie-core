Optional allVariants

allVariants?: boolean
  • Defined in blueprints-integration/src/action.ts:91

Set to true if ad-lib action should can be used in any showstyle-variant. Default: false = only used by the current variant. This is useful for actions in Buckets, so that they can be easily shared between rundowns. Note: When used, this must be set for ALL adlibs of a certain type (not just a few variants). Note: Only valid for items returned from getAdlibItem
