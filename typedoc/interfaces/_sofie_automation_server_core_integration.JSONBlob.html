Interface JSONBlob<T>

JSONBlob is intended to be used instead of JSON.parse/JSON.stringify, in order to share stricter typings between the stringifying (sending) and the parsing (receiving) side.

Usage: interface A { a: number } const aBlob: JSONBlob = JSONBlobStringify({ a: 1 }) const a: A = JSONBlobParse(aBlob)
